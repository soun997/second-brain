## 서론

t3.medium과 t3.small 인스턴스의 차이는 메모리의 크기이다.
t3.medium은 4GB, t3.small은 2GB의 메모리를 가지고 있다.

연산은 CPU의 역할일텐데, 메모리가 어떤 차이를 만드는지에 대해 궁금하였다.

이 둘은 API 요청에 대하여 다른 퍼포먼스를 보여줄 수 있을까?
